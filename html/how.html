How It's Made

Project Overview

This project was created to demonstrate the process of building a portfolio web from scratch. It includes various components such as HTML, CSS, and JavaScript to create a fully functional website.

Technologies Used

  • HTML5
  • CSS
  • JavaScript
  • Version Control with Git
  • Visual Studio Code

Development Process

  1. Planning and Design: Initial sketches and wireframes were created to outline the structure and layout of the website.
  2. HTML Structure: The basic HTML structure was created to define the content and layout.
  3. Styling with CSS: CSS was used to style the HTML elements and create a visually appealing design.
  4. Adding Interactivity: JavaScript was added to provide interactivity and dynamic content.
  5. Testing and Debugging: The website was tested across different browsers and devices to ensure compatibility and responsiveness.
  6. Deployment: The final version of the website was deployed to git pages for public access.
